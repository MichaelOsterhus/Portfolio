A PHOTO EDITOR PROJECT

The Photo-Editor

The Goal

The goal is to create a Photoshop-like photo-editing app for the purpose of understanding what's underneath the hood of an image-manipulation machine.

Because the goal is understanding, there may be a lot of reinventing the wheel.

Rather than try to make the editor all at once, the project has been broken up into smaller projects. As the photo-editor will be built with JavaScript, each project is presented as a tutorial to explain the steps in JavaScript.

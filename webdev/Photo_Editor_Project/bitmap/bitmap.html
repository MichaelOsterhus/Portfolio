A Presentation On Bitmap Images

As part of building a photo editor, it is important to understand the thing we will be editing: a bitmap image.

A bitmap image is essentially a digital file that maps a grid of squares called pixels. Each pixel in a bitmap image is meant to correspond to a pixel on a display device such as a computer monitor or smartphone screen but because we can zoom in and out, this isn't always the case. Looking at a bitmap image upclose it seems that each pixel square is assigned a single color, but, in fact it is assigned varying values of 3 colors: red, blue, and green.

The image above is meant to represent a bitmap image though it is not in a bitmap file type. Rather, it is drawn dynamically using JavaScript and an HTML Canvas element so that each square meant to represent a pixel is colored by randomly generated red, green,and blue values within a certain range.

Bitmap images account for most but not all images you see on the internet. Also known as raster images, today they come in many file types such as JPEG, PNG, GIF, and BMP. With bitmap images such as these is not unusual for a single pixel to be given not a bit, but 3 (to 4) bytes of data with one byte for the red, green, blue, (and alpha) values

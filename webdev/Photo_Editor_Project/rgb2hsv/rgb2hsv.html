RGB to HSV JavaScript Tutorial

As part of my photo editor projecct, I wanted to work with converting RGB (red, green, and blue) data to HSV (hue, saturation, value), and vice versa, as a way to understand the relationship.

I've made an HSV adjustment tool using JavaScript that takes input from the sliders. Pixel data in a bitmap image contains only RGB data, sometimes RGBA (alpha = opacity). So, if, say, you want to turn a color image into black and white, i.e.reduce the saturation, you have to do it by manipulating the red, green, and blue values. But what is the relationship between RGB values and grayscale values (value), or hue, or saturation?

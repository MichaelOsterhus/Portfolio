Puzzle App

The Goal

  1. Upload an image
  2. Crop the image using a common denominator for width and height
  3. Extract pixel data on squares of that denominator and put the data in an array
  4. randomly reorder the data for the squares
  5. Create a grid of the randomly reordered squares that a user must drag and drop to reorder to complete the puzzle.

Puzzle App

The Goal

The goal here to is cut an uploaded image into a grid, scramble the pieces, and have the user rearrange them.

Phase 1:

First I wanted to be able to make a grid of draggable squares.

Jumble

Phase 2

uploading an image.

Keeping things simple, I crop the image as a square and put it to a canvas element.

The canvas is then cut up in squares which are then used as the background image for 64 div elements, and put into an array which can be rearranged.

Take the squares which are now in an array, shuffle the array and redraw the squares.

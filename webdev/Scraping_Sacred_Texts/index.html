Scraping Sacred Texts With Python

Scraping texts from sacred-texts.com will require a slightly different approach than scraping books of the Bible by chapter from BibleHub. For one, the index of books at sacred-texts is not standardized like the books of the bible, so it won't be as simple as getting a csv file with book names and number of chapters that we can loop through for changing the URL in our GET requests.

There are many books at sacred-texts and they are grouped by category. Each category is listed on the main page with a link to its own page. So, first we must get the URLs from each link in the category menu and pair it with the category name in a dataframe.

The Python libraries used are:

  • BeautifulSoup: for parsing the HTML returned in the HTTP GET request
  • requests: for making the GET requests
  • pandas: for creating and working with dataframes
In[]:
import pandas as pd
from bs4 import BeautifulSoup
import requests

Fetch the HTML of the main page.

As we will be fetching many URLs throughout the process we define a function 'getSoup' to avoid redundancy.

In[]:
first_url = "https://sacred-texts.com/index.htm"

def getSoup(url):
    # Send an HTTP GET request to the URL
    response = requests.get(url)

    # Parse the HTML content
    return BeautifulSoup(response.text, 'html.parser')

soup = getSoup(first_url)

print('HTML fetched.')

Pull URLs from the category menu into dataframe

We will need to repeat the following process so again we define a function 'getURLS'.

To loop through the URLs or search by index later, we create a dataframe from the output.

In[]:
def getURLS(span, text_column):
    # Find all of the link 'a' elements
    a_tags = span.find_all('a')

    # Initialize lists to store URL and text data
    urls = []
    texts = []
    for a in a_tags:
        href = a.get('href')
        text = a.get_text()
        
        # Check if href is not None (some <a> tags may not have href attributes)
        if href:
            urls.append(href)
            texts.append(text)

    # Create a DataFrame from the extracted data
    data = {'URL': urls, text_column: texts}
    return pd.DataFrame(data)

pd.set_option('display.max_rows', None)

I check only the body as there may be links in the head and there is no reason to get these. If there are, I run getURLS again. Otherwise I print the text of the page.

def getText(url):
    soup = getSoup(url)
    body = soup.find('body')
    
    #unwanted elements are in 'center' elements
    unwanted_elements = body.findAll('center')
    for el in unwanted_elements:
        el.extract()

    #Check for links in the body
    if body.find('a'):
        df = getURLS(body, 'Chapter')
        return df
    else:
        return body
